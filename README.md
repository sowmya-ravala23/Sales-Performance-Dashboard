# Sales Performance Dashboard

## Project Overview

This project analyzes sales data using Power BI to identify trends, regional performance, top-selling products, and key business insights.

---

## Tools Used

- Power BI
- DAX
- Excel
  

---

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value

---

## Dashboard Features

- Sales Trend Analysis
- Region-wise Sales
- Category-wise Sales
- Top Products
- Interactive Slicers
  

---

## Business Insights

- West region generated the highest revenue.
- Technology category contributed the highest sales.
- Sales increased significantly during Q4.
- Top 10 customers generated a large share of total revenue.
